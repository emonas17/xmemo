# Xmemo Screenshot Checklist

App Store usually needs screenshots from a real device or simulator. Best set: 6.7-inch iPhone screenshots.

## Recommended Screens

1. Recording screen
   - Show the red record button and the VU meter.
   - Caption: Record a voice reminder in seconds

2. When to remind screen
   - Show Today buttons: In 1 hr, In 3 hr, In 5 hr.
   - Caption: Choose today, tomorrow, or a custom time

3. Custom date and time
   - Open Date and time calendar/time picker.
   - Caption: Schedule exactly when you need it

4. Reminder playback screen
   - Show play button and snooze buttons.
   - Caption: Listen, snooze, or stop repeating

5. Repeat days screen
   - Show weekday selection and Morning/Afternoon/Evening row.
   - Caption: Repeat on the days you choose

6. Settings screen
   - Show sound, speaker/earpiece, and icon choices.
   - Caption: Pick sounds, icon, and playback style

## iPhone Screenshot Sizes Apple Accepts

- 6.7-inch: 1290 x 2796 px
- 6.5-inch: 1242 x 2688 px
- 5.5-inch: 1242 x 2208 px

If only one set is uploaded, start with 6.7-inch.
