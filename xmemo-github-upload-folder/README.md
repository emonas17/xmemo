# Xmemo

Xmemo is a fast voice reminder app for iPhone. Record a short memo, choose when it should return, and listen to the original recording from a simple reminder screen.

## What Is Included

- `VoiceReminder/` - iOS Xcode project for Xmemo
- `AppStoreAssets/` - App Store text, screenshot checklist, support page, and privacy policy
- `support.html` - GitHub Pages support URL
- `privacy.html` - GitHub Pages privacy policy URL
